deepHPI server is now online

Nov 1, 2019

To bolster and accelerate the research on infectious disease we have implemented deepHPI. The deepHPI web server is a platform that uses Convolutional Neural Networks models (CNN) for the accurate prediction of host-pathogen protein-protein interactions. The server delivers four host-pathogen model types: plant-pathogen, human-bacteria, human-virus & animal-pathogen; this, in order to account for a wide range of analyses and cases of use. Furthermore, it also provides an enriched visualization of the resulting host-pathogen network.
